#cybersecurity #web-security 

Session Hijacking - exploitation of a valid session assigned to a valid user.

The attacker can get the victims session identifier with help of:

- Scripting attack ([[202410312249 Reflected XSS|XSS]])
- Session Prediction
- Session ID Brute Force
- Packet sniffing
- Direct access to file system
- Direct access to browser history

#### Session Hijacking reasons

**Weak Session ID Generation**
**Weak Session ID Maintenance:**
- Insecure Transfer
- Vulnerable Session Termination
- Weak Timeout Mechanism
- Insecure Session Tracking
