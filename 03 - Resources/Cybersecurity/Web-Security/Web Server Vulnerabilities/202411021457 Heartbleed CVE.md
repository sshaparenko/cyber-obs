**Heartbleed** is a security bug discovered in April 2014 in the OpenSSL cryptography library.

It results from improper input validation (due to the missing bounds check) in the implementation of he TLS heartbeat extension.

This vulnerability is classified as buffer over-head, a situation where more data can be read than should be allowed
