**Shellshock** also known as Bushbug\Bashdoor is a family of security bugs in the widely used UNIX Bash shell

An Attacker could exploit a machine running Bash by forcing it to set specifically crafted environment variables. This could then be further exploited to let them execute shell commands i.e., run programs on other people’s computers
